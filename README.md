# PTV-Hydrogen-Model
A photonic toroidal vortex model for calculating hydrogen atom hyperfine levels

A phenomenological geometrical model is presented for the hyperfine structure of hydro-7 genic systems. The approach extends a previously published fine-structure analysis to hy-8 perfine splittings by introducing a compact set of empirically constrained geometric cor-9 rections. Unlike conventional quantum electrodynamic treatments which reference the 10 Lamb shift to the hyperfine centroid, the present framework targets the hyperfine mid-11 point, corresponding to a substantially larger reference interval. Despite this difference in 12 reference scale, agreement with experimental hyperfine and Lamb-shift-related data at the 13 0.01 MHz level is obtained across multiple hydrogen states. The model is further extended 14 to deuterium, tritium, 3He+, and 7Li2+, revealing systematic cross-nuclear behaviour and a 15 universal scaling relation involving nuclear mass number and charge. The calculations 16 employ a simplified geometric scheme that does not rely on perturbative quantum elec-17 trodynamics but is intended as a complementary phenomenological description. Possible 18 physical interpretations in terms of structured internal dynamics are discussed, together 19 with limitations and directions for further development.

This code implements the PTV model as described in Dr Barry R. Clarke's research.
